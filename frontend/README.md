# TLU Scholar Editor

Frontend repository for the TLU Scholar Editor application.

This is an internal university web application designed for students, lecturers, and administrators, providing a professional, academic, and productivity-focused environment.

## Technologies

- React
- Vite
- Tailwind CSS

## Running the project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Guidelines

Please follow the rules established in `guidelines/` when contributing to this project.
