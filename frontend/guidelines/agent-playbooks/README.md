# Agent Playbooks (Project-Local)

These playbooks are project-local equivalents of Codex "skills", so they work in VS Code even if the editor environment does not load `~/.codex/skills`.

Use them as the default rules when generating UI or implementing Figma designs for this repo.

Playbooks:

- `frontend-ui.md`: Visual/hierarchy rules for app surfaces (not landing pages).
- `figma-workflow.md`: Figma Make + Figma-to-code workflow.
- `shadcn-ui.md`: How to use/extend the local shadcn-style UI library in `src/app/components/ui/*`.
- `typst-editor-reference.md`: Reference notes from `typst-online-editor` and `texlyre` for Phase 5 preview/worker/editor work.
